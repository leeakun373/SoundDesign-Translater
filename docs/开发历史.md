## 1. 要做什么

**离线**音效翻译工具，服务三类场景：

| # | 场景 | 输入示例 |
|---|------|----------|
| ① | 描述/说明 | 英文 Soundminer 描述句 |
| ② | 库文件名 | `ICEFric_Ice Axe Scratch ... 01_ASO_Rec_CO100K` |
| ③ | 多工具共用 | UCSRenamer / Reaper 等同进程或 HTTP 调用 |

**已确认决策：**

- 文件名输出 **格式 B**：`CatID_中文词组 01_厂商码_设备码`（码不译）
- 术语人工入口：**`glossary/user_overrides.csv`**（最高优先级）
- 引擎：**NLLB-200 int8 + CTranslate2**，不换模型

## 2. 架构一览

```
┌─────────────────────────────────────────────────────────┐
│  app.py / service/server.py / examples/*                │
└───────────────────────────┬─────────────────────────────┘
                            ▼
┌─────────────────────────────────────────────────────────┐
│  engine.py — 语言检测、模式路由、NLLB 推理              │
└───────┬─────────────────────────────┬─────────────────┘
        │ 英→中                        │ 中→英
        ▼                              ▼
  filename_parser.py              zh_compose.py
  matcher.segment_*               + zh_oral_aliases.csv
  polish.py                       （术语最长匹配组装，不碎句 NLLB）
        │                              │
        └──────────────┬───────────────┘
                       ▼
              audio_glossary.sqlite
              （build_glossary.py 生成）
```

| 模块 | 作用 |
|------|------|
| `engine.py` | NLLB 加载；`translate(text, mode, pro_mode)` |
| `glossary/matcher.py` | 英→中：术语分段、词边界匹配、force_replace |
| `glossary/filename_parser.py` | 文件名拆 `_`/空格；CatID/码保护；格式 B |
| `glossary/zh_compose.py` | 中→英：最长匹配 + 口语表 → FX 英文关键词串 |
| `glossary/polish.py` | 中文后处理；英文 FX 空格/Title Case |
| `glossary/build_glossary.py` | 合并 sources + overrides → SQLite |

## 5. 关键问题与修法（备忘）

| 问题 | 原因 | 修法 |
|------|------|------|
| NLLB 破坏术语占位符 | 占位符被模型改写 | 改为 **分段翻译**：术语段不送 NLLB |
| `ASO.wav` 变「澳门银河」 | 扩展名导致码未识别 | filename_parser 剥 `.wav` 再判 never_translate |
| 中→英输出 `Oh my God` / 乱拼接 | 中文碎句送 NLLB + 无空格 join | **zh_compose** 最长匹配；英文用空格拼 |
| 「房间底噪」→ Conservatory | UCS 底库多 EN 共用同一 ZH | compose 按 **priority** 去重，overrides 优先 |
| 混合输入「关门声 exterior shut」方向错 | 拉丁字母多于汉字 | `detect_lang_pair` 混合文本优先中→英 |
| `Hi`→您好、`FIRING`→机枪 | 文件名普通词误送 NLLB | FILENAME_RESERVED + overrides never_translate |
| Win 端口 8765 被拒 | 权限/占用 | 默认改 **18765** |
| `.bat` 中文注释无反应 | cmd 编码 | 启动脚本改纯 ASCII |

## 6. 翻译行为速查

| 方向 | 条件 | 行为 |
|------|------|------|
| 英→中 文件名 | `pro_mode` + 含 `_`/CatID | filename_parser + 术语 + NLLB 补洞 |
| 英→中 句子 | `pro_mode` | segment 术语替换 + NLLB 普通段 |
| 中→英 | `pro_mode` + 中文为主 | **zh_compose**（命中≥1 即返回，否则整句 NLLB 兜底） |
| 任意 | `pro_mode=false` | 纯 NLLB |

**用户体验要点：** 中→英是 **FX 关键词串**（`Ice Axe Scratch Friction`），不是通顺英文句子。
