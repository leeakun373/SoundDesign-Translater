# 模型与运行配置

---

## 1. 翻译模型（NLLB）

| 项 | 说明 |
|----|------|
| **目录** | `nllb_int8_model/`（项目根，与 `engine.py` 同级） |
| **代码** | `engine.py` → `MODEL_DIR = Path(__file__).parent / "nllb_int8_model"` |
| **来源** | HuggingFace `JustFrederik/nllb-200-distilled-600M-ct2-int8` |
| **获取** | GitHub Release 附件 `nllb_int8_model.zip`，或 `scripts/download_assets.ps1` |

必需文件：`nllb_int8_model/model.bin`（仅 General / 旧版 `python app.py` 需要；FXName Normalize 双击入口不加载 NLLB）。

### 语言代码

| 语言 | 值 |
|------|-----|
| 简体中文 | `zho_Hans` |
| 英文 | `eng_Latn` |

### 设备

自动选择 CUDA / CPU（`engine.detect_device()`），精度 `int8`。

---

## 2. HTTP 服务

| 项 | 默认 | 配置 |
|----|------|------|
| 地址 | `127.0.0.1` | `service/server.py` |
| 端口 | `18765` | 环境变量 `LOCAL_TRANSLATE_PORT` |

---

## 3. 术语库底库 CSV

默认均在项目内（相对路径）：

| 环境变量 | 默认 |
|----------|------|
| `GLOSSARY_MY_KEYWORD` | `glossary/sources/MyKeyWordV1.csv` |
| `GLOSSARY_UCS_CATID` | `glossary/sources/ucs_catid_list.csv` |
| `GLOSSARY_UCS_OFFICIAL` | `glossary/sources/ucs_categorylist.csv` |

输出：`glossary/audio_glossary.sqlite`（Release 亦提供预构建副本）。

---

## 4. 可选：商业库挖词

| 环境变量 | 说明 |
|----------|------|
| `UCS_COMMERCIAL_DB` | 外部 UCS 商业库 `ucs_assets.db` 路径 |

用于 `tools/find_uncovered_terms.py`、`tools/export_uncovered_batches.py`（非运行必需）。

---

## 5. 翻译逻辑入口

| 功能 | 模块 |
|------|------|
| 英→中 文件名 | `glossary/filename_parser.py` |
| 英→中 句子 | `glossary/matcher.py` |
| 中→英 FX 组装 | `glossary/zh_compose.py` |
| 口语别名 | `glossary/zh_oral_aliases.csv` |

GUI「音频专业模式」= `pro_mode=True`。
